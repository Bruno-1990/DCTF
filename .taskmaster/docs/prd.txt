# PRD - Sistema DCTF MPC - Melhorias e Expansão

## 1. Visão Geral do Projeto

O DCTF MPC é um sistema completo de análise e processamento de dados fiscais brasileiros, incluindo DCTF (Declaração de Débitos e Créditos Tributários Federais), SPED (Sistema Público de Escrituração Digital), IRPF e outras obrigações fiscais. O sistema possui arquitetura moderna com backend Node.js/TypeScript, frontend React/TypeScript, integração com Supabase e MySQL, e processamento avançado de documentos fiscais.

## 2. Objetivos do Projeto

### 2.1 Objetivos Principais
- Melhorar a qualidade e confiabilidade do sistema de análise fiscal
- Expandir funcionalidades de validação e correção automática
- Otimizar performance e escalabilidade
- Melhorar experiência do usuário no frontend
- Implementar testes automatizados abrangentes
- Documentar melhor o sistema e suas APIs

### 2.2 Objetivos Técnicos
- Refatorar código legado para padrões modernos
- Implementar arquitetura de microserviços onde apropriado
- Melhorar tratamento de erros e logging
- Implementar monitoramento e observabilidade
- Otimizar consultas ao banco de dados
- Melhorar segurança e validação de dados

## 3. Escopo do Projeto

### 3.1 Módulos Principais

#### 3.1.1 Módulo DCTF
- Processamento e análise de declarações DCTF
- Validação de códigos e estruturas
- Geração de relatórios e dashboards
- Integração com sistemas externos

#### 3.1.2 Módulo SPED v2.0
- Sistema de validação e correção de arquivos SPED (EFD ICMS/IPI)
- Pipeline de processamento em 5 camadas (Normalização, Validação Interna, Matching, Classificação, Correção)
- Sistema RAG para documentos legais
- Packs por segmento (Comércio, Bebidas, Indústria, E-commerce)
- Base de conhecimento com versionamento

#### 3.1.3 Módulo IRPF
- Processamento de dados IRPF
- Cálculos fiscais e relatórios
- Cache e otimizações

#### 3.1.4 Módulo Clientes
- Gestão de clientes e dados cadastrais
- Integração com ReceitaWS para validação de CNPJ
- Sincronização de dados
- Normalização de CNPJ e razões sociais

#### 3.1.5 Módulo Banco de Horas
- Processamento de dados de banco de horas
- Relatórios e análises
- Integração com sistema SCI

#### 3.1.6 Módulo Conferências
- Sistema de conferência de dados fiscais
- Dashboard administrativo
- Relatórios de divergências

### 3.2 Funcionalidades Transversais

#### 3.2.1 Autenticação e Autorização
- Sistema de autenticação seguro
- Controle de acesso baseado em roles
- Middleware de validação

#### 3.2.2 APIs e Integrações
- RESTful APIs bem documentadas
- WebSocket para atualizações em tempo real
- Integração com serviços externos (ReceitaWS, etc.)

#### 3.2.3 Processamento de Arquivos
- Upload e processamento de arquivos (CSV, XLSX, XML, PDF)
- Validação de formatos
- Tratamento de erros
