# Assets - Ícones

Esta pasta deve conter os ícones do aplicativo.

## Ícones Necessários

Para o funcionamento completo do monitor, você precisa dos seguintes ícones:

- `icon.ico` - Ícone principal (16x16, 32x32, 48x48, 256x256)
- `icon-online.ico` - Ícone quando serviço está online (verde)
- `icon-offline.ico` - Ícone quando serviço está offline (vermelho)

## Formato

- Formato: `.ico` (Windows Icon)
- Tamanhos recomendados: 16x16, 32x32, 48x48, 256x256 pixels
- Formato de cor: 32-bit com transparência

## Gerar Ícones

Você pode usar ferramentas online como:
- https://convertio.co/pt/png-ico/
- https://www.icoconverter.com/
- https://favicon.io/

Ou ferramentas desktop:
- IcoFX
- GIMP (com plugin ICO)
- Photoshop

## Ícones Temporários

Se você não tiver ícones customizados, o aplicativo usará ícones padrão do sistema. Para melhor experiência visual, recomenda-se criar ícones personalizados.

## Estrutura Esperada

```
assets/
├── icon.ico          (ícone principal)
├── icon-online.ico  (ícone verde - online)
└── icon-offline.ico (ícone vermelho - offline)
```
